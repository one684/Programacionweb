Divs en Html

es una etiqueta que se utiliza para separar y organizar el contenido de una página web. La palabra div viene del inglés "division" que significa "división", "sección" o "área".
s el contenedor genérico para el contenido. No tiene efecto en el contenido o en el diseño hasta que se le aplica un estilo de alguna manera usando CSS (p. ej. se le aplica un estilo directamente o se aplica algún tipo de modelo de diseño como Flexbox a su elemento principal).
